Code by Edge is a unique luxury jewellery brand that allows the customisation of rings, necklaces, bracelets and earrings with specific messages through Morse code. For this project we designed and developed an amazing website, a state-of-the-art 3D customiser (first of its kind) and a retailer platform. Everything was possible by using the latest web technologies, such as React Native, powered by a full-featured backend system based in WordPress.
